These scripts will download and/or create DeepOnion.conf for DeepOnion Wallet. 
China users or other Tor blocked countries will use linuxc.sh . 
All others will use linux.sh .

Before running script you will need to make it executable.
In a terminal type:

chmod +x linux.sh

OR

chmod +x linuxc.sh

depending on which you are using. 

To execute type:

./linux.sh

OR

./linuxc.sh
